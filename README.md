# Selah Display System

A Raspberry Pi 5-based digital photo frame system for family media, artwork, verses, calendar events, and more.

## Prerequisites

- Raspberry Pi 5 with Raspberry Pi OS (Bookworm).
- Raspberry Pi Camera Module (motion detection).
- USB microphone (voice control, e.g., USB webcam mic or dedicated USB mic).
- Touchscreen (optional, e.g., Raspberry Pi 7" touchscreen).
- Internet connection.
- A dedicated Gmail account for the display (e.g. `your-display@gmail.com`).
- Google API credentials (Calendar/Drive).
- OpenWeatherMap API key (weather).
